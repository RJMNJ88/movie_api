Movie API

What this API if used for:

This "Movies API" will function as a database of different movies, directors, actors, and genres. Users will be able to create their own personal accounts and create lists of their favorite movies, which will be saved to the database for later use an reference by the user.

Request URL HTTP Method URL Parameters Request Body Format Response Body Format
Return all movies /movies GET /movies None A JSON object containing a list of movies
Return a single movie /movies/[title] GET /movies/:title None A JSON object containing data about a single movie:
{
title: 'The Hobbit: An Unexpected Journey',
genre: ['Fantasy'],
released: 2012,
director:
{
name: 'Peter Jackson',
born: 'October 31, 1961',
country: 'New Zealand'
}
}
Return movies by genre /genre GET /genre None A JSON object containing a list of movies
matching a particular genre
Return movies by director /director GET /director None A JSON object containing a list of movies
by a particular director
Register new user /users POST /users A JSON object holding data about the
user, structured like:
{
username: "exampleUser",
password: "examplePassword",
email: "exampleEmail@email.com"
} 		A text response indicating that the new user
has been successfully registered.
Update user info /users/[userinfo] PUT /users/:userinfo None A text response indicating that the user info
has been successfully updated.
Add movie to favorites /users/[userinfo]/movies/[title] POST /users/:userinfo/movies/:title A JSON object holding data about the
user, structured like:
{
title: 'The Hobbit: An Unexpected Journey',
genre: ['Fantasy'],
released: 2012,
director:
{
name: 'Peter Jackson',
born: 'October 31, 1961',
country: 'New Zealand'
}
} 		A text response indicating that the movie was
successfully added.
Remove movie from favorites /users/[userinfo]/movies/[title] DELETE /users/:userinfo/movies/:title None A text response indicating that the movie was
successfully removed.
Delete user account /users/[userinfo] DELETE /users/:userinfo None A text response indicating that the user
account had been deleted.
